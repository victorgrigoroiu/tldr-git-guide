### Inititalize repository

```
$ git init
```

### Clone existing repository

```
$ git clone URL
```

### Add file to repository

```
$ git add ./path/to/folder
```
### Remove files staged for a git commit

```
$ git reset path/to/file
```

### Show staging area

```
$ git status
```

### Show commit history

```
$ git log --graph --decorate --oneline
```
